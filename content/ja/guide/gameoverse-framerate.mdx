### FPS に影響を与える主な要因

| 要因 | FPS への影響 | 最適化戦略 |
| :---------------- | :-------------------------------------------- | :------------------------------------------------------ |
| **グラフィックスカード** | レンダリング性能の主要な決定要因 | ドライバーの更新、解像度/設定を下げる、オーバークロック（安全な場合） |
| **プロセッサ (CPU)** | ゲームロジック、AI、物理演算を処理 | バックグラウンドアプリを閉じる、十分な冷却を確保 |
| **RAM (メモリ)** | ゲームアセットとデータを保存 | 十分なRAM（16GB以上推奨）を確保、メモリを大量に消費するアプリを閉じる |
| **ストレージ (SSD/HDD)** | ロード時間に影響、FPSへの直接的影響は少ない | 高速ロードのために Gameoverse を SSD にインストール |
| **ゲーム内設定** | 視覚的な複雑さを直接制御 | 解像度、テクスチャ品質、影、アンチエイリアスを調整 |

Gameoverse のフレームレートを改善するための最も即効性があり、影響力のある方法は、ゲーム内のグラフィック設定を調整することです。Gameoverse は、視覚的な品質とパフォーマンスのバランスをとるために微調整できる幅広いオプションを提供しています。システムごとに特定の設定に対する反応が異なるため、実験が鍵となります。

<Callout type="warning" title="「ウルトラ」設定に注意">
「ウルトラ」設定は最高のビジュアルを約束しますが、多くの場合、大きなパフォーマンスコストを伴います。最高級のゲーミングリグを持っていない限り、安定したフレームレートを得るには「高」と「中」の設定のバランスを目指すのが最も効果的なアプローチです。
</Callout>

### 推奨される設定調整

<Steps>
  <Step num="1" title="解像度を下げる">
    ディスプレイ解像度を下げることは、FPS を向上させる最も効果的な方法の 1 つです。1080p 解像度（1920x1080）は、視覚的な明瞭さとパフォーマンスの間の良い妥協点となることが多いです。モニターが 1440p や 4K の場合は、1080p に下げることで大幅な FPS 向上が期待できます。
  </Step>
  <Step num="2" title="テクスチャ品質を調整する">
    テクスチャ品質は、ゲーム内の表面の詳細に影響します。高品質のテクスチャは見栄えが良いですが、多くの VRAM を消費します。グラフィックスカードの VRAM が 8GB 未満の場合は、これを「ウルトラ」ではなく「中」または「高」に設定することを検討してください。
  </Step>
  <Step num="3" title="影の品質を下げる">
    影は GPU に非常に負荷をかけることで知られています。影の品質を下げたり、一部の影エフェクトを無効にしたりすることで、視覚的な影響を比較的抑えつつ、フレームレートを大幅に向上させることができます。
  </Step>
  <Step num="4" title="アンチエイリアス (AA) を無効にする">
    アンチエイリアスはギザギザのエッジを滑らかにしますが、リソースを大量に消費する可能性があります。MSAA の代わりに FXAA のような負荷の低い形式を使用するか、無効にしてみてください。最近のゲームには、バランスの良い優れたテンポラルアンチエイリアス（TSSAA）が搭載されていることが多いです。
  </Step>
  <Step num="5" title="描画距離を下げる">
    この設定は、遠くのオブジェクトがどこまでレンダリングされるかを決定します。これを下げると、特にオープンワールドエリアで FPS が向上しますが、オブジェクトが不自然にパッと現れる（ポップイン）原因になる可能性があります。
  </Step>
</Steps>

### パフォーマンス vs. 視覚的な忠実度

| 設定 | パフォーマンスへの影響 | 視覚的な影響 | 推奨アクション |
| :---------------------- | :----------------- | :------------ | :--------------------------------------------------- |
| **解像度** | 高 | 高 | 高解像度で苦戦する場合は 1080p に下げる |
| **テクスチャ品質** | 中 | 中 | 4-6GB VRAM なら高/中、8GB 以上ならウルトラ |
| **影の品質** | 高 | 中 | 大幅な FPS 向上のために中または低 |
| **アンチエイリアス** | 高 | 中 | 無効にするか FXAA/TSSAA を使用 |
| **描画距離** | 中 | 中 | 中から高、オープンワールドのパフォーマンスに基づいて調整 |
| **ポストプロセス** | 中 | 中 | ブルーム、モーションブラー、被写界深度などのエフェクトを下げる |
| **ボリュメトリックライティング** | 高 | 高 | 無効にするか低に設定 |

</Section>

### ドライバー設定の推奨事項

ほとんどのグラフィックスカードコントロールパネル（NVIDIA コントロールパネル、AMD Radeon Software）には、パフォーマンスのために微調整できる追加の設定があります。

| 設定 (NVIDIA/AMD) | 推奨値 | メリット |
| :-------------------- | :------------------ | :---------------------------------------------- |
| **画像のシャープニング** | オフ | 追加のレンダリング負荷を防止 |
| **低遅延モード** | ウルトラ / 有効 | 入力遅延を減らし、応答性を向上させる可能性 |
| **電源管理** | パフォーマンス最大化を優先 | GPU がフルパワーで動作することを保証 |
| **テクスチャフィルタリングの品質** | パフォーマンス / 高パフォーマンス | テクスチャの詳細よりも FPS を優先 |
| **垂直同期 (V-Sync)** | オフ（ティアリングが問題ない限り） | 入力遅延を減らし、リフレッシュレート以上の FPS を許容 |

</Section>

### 高度な最適化戦略

1.  **オーバークロック：** CPU と GPU を慎重にオーバークロックすることで、大幅なパフォーマンス向上を得ることができます。ただし、リスクを伴うため、プロセスと潜在的な影響を理解している場合にのみ試みてください。常に信頼できるガイドを調査し、それに従ってください。
2.  **RAM 速度：** 高速な RAM（例：DDR4-3200MHz 以上）は、特に CPU がボトルネックとなっているシナリオで CPU パフォーマンスを向上させることができます。マザーボードの BIOS/UEFI 設定を介して、RAM が公表されている速度で動作していることを確認してください。
3.  **将来のハードウェアアップグレード：** すべてのソフトウェア最適化で満足のいく結果が得られない場合は、ハードウェアのアップグレードが必要になる可能性があります。予算と現在のシステムのボトルネックに基づいて、まずグラフィックスカードのアップグレードを優先し、次に CPU、最後に RAM の順で検討してください。
