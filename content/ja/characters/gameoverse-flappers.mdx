<Section icon={Target} title="フラッパーズの進化：オリジナル版とリブート版のデザイン">

| 特徴       | オリジナルシリーズ（2009年）                             | リブートシリーズ（2026年）                                  |
| :------------ | :------------------------------------------------- | :------------------------------------------------------ |
| **全体的な外観** | シンプルに描かれた、鮮やかな青色の丸いイルカ。          | 非常に似ているが、洗練されたディテール。              |
| **目**      | シンプルな点。                                       | 丸くて漫画のような、**青い虹彩**の目。            |
| **口**     | 通常は詳細に描かれていない。                            | 口と**歯列**が描かれている。              |
| **顔**    | なし。                                              | **ピンクの赤面**が特徴。                              |
| **背びれ** | 標準サイズ。                                     | オリジナルデザインよりも**大きい**。                    |
| **筋肉** | あまり強調されていない。                                   | 戦闘中に時折**筋肉や胸筋**が増強される。  |
