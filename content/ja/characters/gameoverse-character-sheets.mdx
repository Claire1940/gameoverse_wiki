<Section icon={Swords} title="Syntax 組織：ヴィランプロフィール">

Syntax 組織は、Gameoverse における主要な脅威として機能します。彼らの **Gameoverse キャラクターシート**は、「生存」ではなく「回収（ハーベスティング）」に焦点を当てているため、独特です。威圧的な Warrick に率いられた Syntax メンバーは、疑うことを知らないヒーローたちのために「解説妖精」や「チュートリアルガイド」として振る舞い、最終的に世界を破壊することになる勝利へと彼らを導きます。

<Section icon={Zap} title="キャラクターの統計とメカニクスの分析">

**Gameoverse キャラクターシート**を確認すると、異なる勢力間で特定の繰り返されるメカニクスが現れます。これらのメカニクスは、キャラクターが宇宙の「ゲームルール」とどのように相互作用するかを定義します。

1.  **Float（フロート）**: 完了したゲーム世界から回収されるエネルギー。Syntax はこれを使用して Gameoverse の「残酷なルール」を書き換えようとしています。
2.  **合体メカ（Combining Mecha）**: Malice や Mayhem のようなロボットキャラクターが使用する凄惨なプロセス。Farcade のアーマー変形とは異なり、Mayhem はより強力なマシンに合体するために Malice を「食べる」必要があります。
3.  **チュートリアルロール**: Miss Information や Fold のようなキャラクターは、ゲーム内のヘルパーとして機能します。Fold は物理的な攻略シートとして機能し、Miss Information は一本調子で繰り返される指示を提供します。

| メカニクス | Farcade での使用 | Syntax での使用 |
| :--- | :--- | :--- |
| **変身** | 英雄的なパワードスーツ (Kaboodle) | 凄惨な共食い (Mayhem) |
| **情報** | 戦略的計画 (Dusk) | 欺瞞的なチュートリアル (Miss Info) |
| **目的** | 完了の阻止 | 完了の強制（回収） |

<Callout type="warning" title="パワーダイナミクス">
Mayhem が Malice と合体するプロセスは「明らかに苦痛」であると記されており、Kaboodle が Kit のアーマーへとシームレスかつ合意の上で変身するのとは対照的です。
</Callout>

</Section>
