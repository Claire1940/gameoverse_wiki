<Section icon={Map} title="Gameoverseストーリーの核心的な仕組み">

**Gameoverseのストーリー**の根幹は、古典的なゲームの定石を覆すことにあります。ほとんどのメディアでは、プレイヤーはヒーローが悪役を倒すことを応援します。しかし、Gameoverseにおいて「クリア条件（Win Condition）」は壊滅的なイベントです。ヒーローがその宿命を果たすと、ゲーム世界は完結を迎え、惑星の爆発を伴う「収穫（ハーベスト）」シークエンスが発動します。

**動画の見どころ:**
- Farcadeの偵察船とその乗組員の紹介。
- 「クリア条件」のメカニズムの発見。
- 「役に立たない」ゲームキャラクターの安全を巡るキッドの葛藤。
- 清掃員「Syntax」の初登場。

キッドが働く組織「Farcade」は、これらの世界を監視しています。彼らの公式な立場は、世界が完結したら、その「ブロート」（残留データやエネルギー）を回収しなければならないというものです。これにより、エージェントたちは実質的に文明の死を待つハゲタカのような存在となり、道徳的な空白が生じています。しかし、キッドはFarcadeが使い捨てと見なすキャラクターを救おうとすることで、標準的なプロトコルから逸脱していきます。

| 用語 | 定義 | ストーリー上の役割 |
| :--- | :--- | :--- |
| **クリア条件** | ゲーム世界の最終目的 | 惑星の爆発を引き起こす |
| **ブロート** | 完結したゲームの残留エネルギー | Farcadeが収穫する資源 |
| **Farcade** | エージェントのマルチバース組織 | ゲーム世界の監視と収穫を行う |
| **Syntax** | ウイルスのような紙状の存在 | 破壊的な清掃員として活動 |
| **ラーナウルス** | 急速な学習が可能な種族 | ゴブルズの種族。偵察を支援 |

<Callout type="warning" title="収穫のルール">
**Gameoverseのストーリー**において、Farcadeは世界を救いたいわけではありません。彼らの主な目的は、ヒーローが悪役を殺すのを手助けしてでも、ゲームを完結させて「ブロート」エネルギーを集めることです。
