<Section icon={Target} title="ゲームバースのルール">

パイロット版は、これらの世界がどのように機能するかを規定する魅力的な一連のルールを紹介します。すべてのゲーム世界には「ヒーロー」と「悪役」がいます。ヒーローが悪役を倒すと、ゲームは「クリア」され、世界は本質的にリセットされるか、「フラックス」（結晶）に爆発します。ファサードの使命は驚くほど直感的ではありません。彼らは、ヒーローが勝つことを確実にしなければ、ゲーム世界とその住民の破壊を意味しても、これらの結晶を収集できません。

これはカイに道徳的なジレンマを生み出します。彼は、ヒーローを助けて世界を「救う」ことが、実際にはその世界の最終的な破滅につながることに気づきます。パイロット版は、「コーラルリーフシティ」編でこの点を強調しています。そこではイルカのフラッパーズがサメのスナッピーを倒すように操作され、壊滅的な惑星爆発を引き起こします。

| 用語 | 定義 | 影響 |
| :--- | :--- | :--- |
| **フラックス** | 結晶化したエネルギー源 | 次元間移動を可能にする |

| **ゲームクリア** | 勝利条件 | 世界の収穫をトリガーする |
| **サイドクエスト** | 不要なタスク | 収穫プロセスを遅らせる |
